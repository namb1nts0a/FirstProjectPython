                =======================Bienvenue dans le Jeux Snake=============================



Ce jeux basé sur le jeux serpent: on doit manger du pomme pour grandir,
on eviter de dépasser la limite du jeux et de manger son corps.
Chaque nouveau meilleur score est sauvgardé afin qu'on a un nouveau record. 

Le source des images et les effets sonores sont dans les sites libre et gratuit d'internet 

Le développement du jeux est fait avec :
        le système linux Parrot 4.19.0 
        l'IDE Pycharm-community-2020.1 
        la bibliothèque graphique 2D de python3 Pygame 1.9.6

-*->C'est seulement du code source mais pas encore compilé. 

Libre a vous de l'améliorer de le modifier et de le partager avec tout le monde.
